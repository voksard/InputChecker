# InputChecker

Minecraft Forge mod (1.8.9) to check input sequences tick by tick.

## Features
- Tick-based input checking
- Lenient / Ignore inputs
- FullSprint mode
- HUD & Statistics panel

## Installation
1. Download the latest `.jar` from Releases
2. Put it in `.minecraft/mods`
3. Launch Minecraft 1.8.9 with Forge

## Commands
- /inputchecker help
